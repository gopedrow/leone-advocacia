# Imagens do site

Coloque aqui os arquivos de imagem usados pela plataforma.

Arquivos esperados (substitua os placeholders):

- `advogada-hero.jpg` — foto profissional da advogada (Hero da home), preferir 1200×1600px (retrato).
- `advogada-sobre.jpg` — foto para a seção "Sobre".
- `og-default.jpg` — imagem Open Graph para compartilhamento (1200×630px).
- `logo.svg` — logotipo (opcional; há um logo tipográfico por padrão).

> A foto enviada na conversa deve ser salva como `advogada-hero.jpg` nesta pasta.
> Formatos recomendados: `.jpg`/`.webp` otimizados. O Next.js fará a otimização automática.
